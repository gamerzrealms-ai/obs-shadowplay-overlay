# 🎬 OBS Notification Overlay

A customizable OBS Studio script that displays clean, animated notifications when recording or replay events occur, it only visible to you do not show up in recordings.

## ✨ Features

* Recording started / stopped notifications
* Replay buffer notifications
* Custom icons (PNG, JPG, GIF)
* Smooth animations and positioning
* ShadowPlay-style visual mode

## 🛠 Requirements

* OBS Studio (with Python support)
* Python 3.x

## 🚀 Installation

1. Open OBS
2. Go to **Tools → Scripts**
3. Click **+** and select `obs_overlay02.py`

## ⚙️ Configuration

* Add your own icons
* Adjust position, animation, size
* Customize overlay behavior
